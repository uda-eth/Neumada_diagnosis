### **3Ô∏è‚É£ Jest May Not Detect Tests in `tests/` Folder**
#### **Problem:**  
By default, Jest looks for test files in `__tests__/`, `*.test.ts`, or `*.spec.ts` inside `src/`.  
If your tests are in `tests/server/`, Jest might **not be configured** to look there.

#### **Fix:**  
Modify **`jest.config.ts`** to explicitly include the `tests/` directory:  

```ts
export default {
  preset: "ts-jest",
  testEnvironment: "node",
  testMatch: ["<rootDir>/tests/**/*.test.ts"],
  collectCoverage: true,
  coverageDirectory: "coverage",
};
```
‚úÖ This tells Jest to run **all tests inside `/tests`**.
